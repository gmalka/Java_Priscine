For Compile program use:
javac -d ./target/ src/java/edu/school21/printer/*/*.java

For Launch app use:
java -cp  ./target Java.edu.school21.printer.App.App <Symbol for empty space> | <Symbol for filled space> | <Path to file>

Example:
java -cp  ./target Java.edu.school21.printer.App.App 0 1 it.bmp
